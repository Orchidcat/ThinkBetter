## 安装
第一种方式：
```
lsusb
ls /dev/video //插入摄像头应该有显示
```


第二种方式：
```
sudo apt-get install fswebcam

//安装完成后运行，可在/home/pi目录下生成拍摄到的实时照片
fswebcam --no-banner -r 640*480 camera.jpg
```
