#python 







- python 只有两种类型的数字，integer，float
- 不支持数字和字符串的混合计算
- list 可以包含任意数量的变量
- 字符串格式化 %用来格式化元组（一个固定大小的list) %s 文本，%d 整数，%f 浮点数
