# 数学第一性学习法  
**——把数学拆到“原子级”：一个问题 = 一个模型 + 一个求解链**  
北京中考 100 分，**本质 = 抽象建模 + 严密推导**

---

## 一、第一性拆解：数学的底层能力矩阵

| 能力维度 | 最基本原子 | 中考题型 | 为什么考？ |
|----------|------------|----------|------------|
| **建模** | 现实 → 数/式/图/关系 | 应用题、函数图象 | 世界是模型 |
| **推导** | 逻辑链：定义 → 性质 → 结论 | 证明、计算 | 宇宙是严密的 |
| **计算** | 变形、化简、求值 | 填空、计算 | 精确是底线 |

**学习组织原则**：**先建模 → 后推导 → 再计算**（金字塔）

---

## 二、学习组织框架：三层金字塔（每天 ≤ 60 分钟）

```
第3层：计算闭环（20 min）
    ↑
第2层：推导链（20 min）
    ↑
第1层：建模拆解（20 min）
```

---

## 第1层：建模拆解（每天 20 min）  
**目标**：**看到题，30 秒内画出“模型图”**

### 动作闭环（3 步）
| 步骤 | 动作 | 工具 |
|------|------|------|
| **1. 画模型** | 每天 1 题 → **数轴 / 坐标 / 表格 / 关系式** | A4 白纸 |
| **2. 标变量** | 圈出 **x, y, a, b, k…** | 高亮笔 |
| **3. 写关系** | 一句话：“y 随 x 的二次变化” | 口述录音 |

**高频模型清单（先画会 10 个）**：
1. 行程：路程 = 速度 × 时间  
2. 工程：工作量 = 效率 × 时间  
3. 浓度：溶质 = 浓度 × 溶液  
4. 利润：利润 = 售价 – 成本  
5. 几何：周长/面积/体积公式  
6. 函数：y = kx + b（一次）  
7. 二次：y = ax² + bx + c  
8. 统计：平均数/中位数/众数  
9. 概率：P = 有利/总  
10. 圆：πr², 2πr, 弧长

> **第一性标准**：**模型图 + 关系式 < 1 分钟**

---

## 第2层：推导链（每天 20 min）  
**目标**：**从模型到结论，写出“每一步因为什么”**

### 动作闭环（3 步）
| 步骤 | 动作 | 工具 |
|------|------|------|
| **1. 列性质** | 每天 1 个核心性质 → **写定义 + 公式** | 卡片 |
| **2. 套模型** | 用第 1 层的图 → **代入性质** | 笔记本 |
| **3. 写推导** | 格式：**因为… → 所以…** | 录音/纸 |

**核心性质卡片（正反面）**：
| 正面 | 背面 |
|------|------|
| 直角三角形 | 勾股定理 a²+b²=c² |
| 相似三角形 | 对应边成比例 |
| 二次函数顶点 | x = -b/(2a) |
| 概率互斥 | P(A+B)=P(A)+P(B) |

> **第一性标准**：**推导链每步有“因为”**

---

## 第3层：计算闭环（每天 20 min）  
**目标**：**3 步万能计算法，任何题 ≤ 3 分钟**

### 3 步计算法（中考模板）
```
① 写已知：列变量 + 数值
② 找关系：写模型公式
③ 变形算：化简 → 代入 → 结果
```

| 动作 | 示例 |
|------|------|
| **每日 1 题** | 二次函数 y=x²-2x-3，求顶点 |
| **步骤** | ① a=1,b=-2,c=-3<br>② 顶点 x=-b/(2a)<br>③ x=1, y=-4 |

**计算闭环**：
1. **先心算**  
2. **再规范写**（带单位/根号/分数）  
3. **查合理性**（结果是否符合模型）

> **第一性标准**：**步骤完整 + 化简到底 + 验算**

---

## 三、每周闭环：实战模拟（周六 90 分钟）

| 项目 | 动作 | 标准 |
|------|------|------|
| **建模** | 画 3 个新模型图 | 每图关系式正确 |
| **推导** | 证明 1 道几何 | 每步有“因为” |
| **计算** | 3 道压轴题 | 每题≤3 分钟 |
| **错题重拆** | 改错 + 重画模型 | 找出“模型错在哪” |
