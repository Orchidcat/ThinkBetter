### **一、现代文阅读母版（记叙文/散文/小说）**

| **题型** | **母版答案模板** | **答题要点** |
|---------|------------------|-------------|
| **1. 概括文章内容/段落大意** | 本文/段通过……（事件/景物/人物），写了……（主要内容），表达了作者……（思想感情）。 | 谁+干什么+结果+感情 |
| **2. 分析词语含义/妙处** | “××”原指……，这里指……，生动形象地写出了……（人物心情/景物特征等）。 | 表面义+语境义+表达效果 |
| **3. 品味句子（修辞/表达方式）** | 运用了……（比喻/拟人/排比等）手法，将……比作……，生动形象地写出了……，表达了……感情。 | 手法+内容+效果+感情 |
| **4. 分析人物形象** | ××是一个……（性格1）、……（性格2）的人。如……（结合具体事例分析）。 | 概括+举例+引文 |
| **5. 理解文章主题/中心思想** | 本文通过……（主要事件），告诉我们……（道理），表达了作者……（赞美/批判等）的情感。 | 事件+道理+感情 |
| **6. 开放性/启发性问题** | 我认为……（个人观点）。因为……（结合文本+生活实际）。 | 文本依据+合理推断 |

### **二、文言文阅读母版**

| **题型** | **母版答案模板** | **答题要点** |
|---------|------------------|-------------|
| **1. 翻译句子** | ××：……（直译）。大意：……（意译）。 | 字字落实+通顺 |
| **2. 解释加点词** | ①……（古义）②……（今义）③……（通假/古今异义/词类活用） | 古义+例句 |
| **3. 断句** | 根据……（标志词：曰、之、而、乎等）断为：…… | 找对话、找虚词 |
| **4. 内容概括** | 文章写了……（事件），表现了……（人物品质）。 | 事件+品质 |
| **5. 分析人物品质** | ××具有……（如智勇双全、忠君爱国等）的品质。文中……（举例）即可证明。 | 品质+事例 |
| **6. 对比课内文言文** | 与《××》中……（相似点）相似，不同的是……（区别）。 | 找共性+个性 |

|--------|-------------|
| **仿写** | 运用……（与例句相同修辞），写出……（内容对称、情感一致）。 |
| **续写** | 承接上文……（情节），运用……（描写手法），突出……（主题）。 |
| **缩写** | 保留……（主要人物+事件+结局），删减……（次要情节）。 |

---

### **使用技巧**：
1. **先读问题，再带着问题读文段**，标注关键词。
2. **答案在原文找依据**，尽量**引用原文**（2-3句为宜）。
3. **字数控制**：简答题50-80字，赏析题80-120字。
4. **规范书写**：分点作答，条理清晰。
