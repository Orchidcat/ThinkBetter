道德经中的圣人，指的是悟道的统治者。

道法自然，万物保持自己的样子。道就在里面显现出来。 

讲了4个方面：
1. 天下大乱（20%）
2. 如何解决（圣人，50%）
3. 道是什么（20%）
4. 德是什么（10%）



### 第一章
```
道，可道，非常道。名，可名，非常名。无名，万物之始；有名，万物之母。故常无欲，以观其眇；常有欲，以观其徼。此两者同出而异名，同谓之玄。玄之又玄，众眇之门。
```

道不可说，名不确定。名称未定之前，万物起源，名称已定，就是万物的母体。

道分两种，我们谈论的道，以及永久的道。人类思考就需要概念与名称。凡是可以说的名称，就不能像道一样永恒。有名，就是人类给万物一个名字的时候。**名是人的理性所需要的**。


无欲，观察发展，而不进行干扰。有欲，就有设计想法，会进行干扰。徼，就是万物的边界。


### 第二章
```
天下皆知美之为美，斯恶已。皆知善之为善，斯不善已。故有无相生，难易相成，长短相较，高下相倾，音声相和，前后相随。是以圣人处无为之事，行不言之教；万物作焉而不辞，生而不有。为而不恃，功成而弗居。夫唯弗居，是以不去。
```

美与恶相对。都是人对价值方面的判断。 有型之物与无形之物。难易，长短，高下，声音，前后。

人的认知活动，很容易产生不同的结果。 如果要做判断，希望从不同的角度来看待问题。

无为：无心而为。没有刻意的目的。
不言：让人不要执着于名称与价值对比上。

正因为不居功，功就不会离开。（做该做的事，而不居功）

生而不有，比如孩子。有其自己的人生要负责。


### 第3章

```
不尚贤，使民不争；不贵难得之货，使民不为盗；不见可欲，使心不乱。是以圣人之治，虚其心，实其腹，弱其志，强其骨。常使民无知无欲。使夫知者不敢为也。为无为，则无不治。
```

老子是不是主张愚民政策？

推贤，就容易产生竞争。什么是难得的东西，就会想去获取。

心思单纯，肚子要吃饱，意志弱一些，不要非做什么不可，身体要健康。
无知无欲： 目的是为了让老百姓安居乐业。
	知 有三个层次：用认知来区分，用认知来避难，用认知来启明

不推崇，不重视，不展示


### 第4章

```
道冲而用之或不盈。渊兮似万物之宗。挫其锐，解其纷，和其光，同其尘。湛兮似或存。吾不知谁之子，象帝之先。
```

空虚而作用，没有极限。像是万物的本源（时间上的最早）。

一定要描写道的话，最好用疑似法。而不是西方那种否定法。

老子在努力描述给世人以道.


### 第5章

```
天地不仁，以万物为刍狗；圣人不仁，以百姓为刍狗。天地之间，其犹橐龠乎？虚而不屈，动而愈出。多言数穷，不如守中。
```

橐龠：tuo2yue4，风箱

天地之间，万物在生长。
天地没有任何偏爱。万物具有荣枯。圣人也没有偏爱，百姓也具有荣枯。

天地之间像风箱一样，一鼓动，就源源不绝。

议论太多，反而无所适从。
中代表虚境。道永远守住虚境的原则。



### 第6章

```
谷神不死，是谓玄牝。玄牝之门，是谓天地根。绵绵若存，用之不勤。
```

生殖力是天地的根本。好像存在，作用无穷无尽。

谷神：山谷，空虚，低卑。神，就是鬼神。神妙的作用。
	使谷成为谷的力量，就是神妙的作用。

比喻道的作用。说道永远存在。


### 第7章

```
天长地久。天地所以能长且久者，以其不自生，故能长生。是以圣人后其身而身先；外其身而身存。非以其无私耶？故能成其私。
```

任何东西，如果求自己生存，就会有竞争出现。 

结果要跟目的区分开。
尽力而为就好。 
设定目的，很容易使用各种手段。就容易变成有偏差的认知，就有偏差的行为。

教育：
过程胜于目的，身教胜于言教（不言之教）。
陶渊明 —— 勤靡余劳,心有常闲

### 第8章

```
上善若水。水善利万物而不争，处衆人之所恶，故几于道。居善地，心善渊，与善仁，言善信，正善治，事善能，动善时。夫唯不争，故无尤。
```

万物跟人类的思维角度。 但这些都有生灭。道就高于这些。2+1 的思维方式。 

水的7种善：
1. 保持谦卑的心态；不要勉强别人
2. 心思善于深沉
3. 总是给人以方便
4. 反应实情（照见须眉）；说话要恰到好处。
5. 清明而有效率；水是平面的，公平的。
6. 做事善于生效
7. 行动善于待时，依时而行。（水有三态）

不想争，所以就没有人能责怪你。 



### 第9章

```
持而盈之，不如其已；揣而锐之，不可长保。金玉满堂，莫之能守；富贵而骄，自遗其咎。功遂身退天之道。
```


咎：就是人和各，代表各说各话。
易经说无咎，就是没好也没坏。就已经很不错了。 

参考：**陶朱公**（范蠡）

看任何问题，从永恒的角度来看。 好，就需要珍惜，不好，就要保守。 


### 第10章

```
载营魄抱一，能无离乎？专气致柔，能婴儿乎？涤除玄览，能无疵乎？爱民治国，能无为乎？天门开阖，能为雌乎？明白四达，能无知乎？生之、畜之，生而不有，为而不恃，长而不宰，是谓玄德。
```

对前面的简单总结。

涤除玄览：心如镜，没有瑕疵。（至人，用心如镜）
天门开阖：收敛自己，雌服
明白四达：不要用知巧
生之、畜之：不限制，不禁止。
**生而不有，为而不恃，长而不宰**： 生了一个东西，不要控制他；做成了一件事，不要仗势自己；做了领导，不要宰制他人。
